# Oasis Infobyte Internship – Data Analytics

This repository contains the projects completed as part of the Oasis Infobyte Data Analytics Internship.

## Completed Tasks

### Task 1 – Exploratory Data Analysis (EDA) on Retail Sales Data

Objective:
Analyze retail sales data to identify trends, top-performing categories, and business insights.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Key Insights:
- Sales by category
- Top-performing states
- Monthly sales trends
- Profit distribution
- Correlation analysis

---

### Task 2 – Customer Segmentation Analysis

Objective:
Analyze customer purchasing behavior to identify valuable customers.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn

Key Insights:
- Top customers by sales
- Top customers by profit
- Customer order frequency

---

### Task 3 – Data Cleaning

Objective:
Prepare the dataset for analysis by improving data quality.

- Saved cleaned dataset
